# Changelog

## osprey-rules 0.1.0

Initial release of the Osprey Rules plugin for Claude Code.

**New:**
- Agent: `osprey-rule-writer` for routing all Osprey SML rule tasks
- Command: `/osprey-validate` for running `osprey-cli push-rules --dry-run`
- Skill: `writing-osprey-rules` for full rule authoring workflow
- Skill: `osprey-sml-reference` for SML type system, labeling patterns, and naming conventions
- Skill: `debugging-osprey-rules` for error diagnosis and fix patterns
